## Read Methods

### Max Stake Duration

Returns the maximum staking duration you can commit for, in seconds. (Applies to [Reward Multiplier](../../apps/staking/reward-multiplier.md))

```solidity
function MAX_STAKE_DURATION() external view returns (uint256);
```
